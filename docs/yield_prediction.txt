# Oil Palm Yield Prediction & Ripeness Guides

## 1. Bunch Ripeness Classification System

### Ripeness Stages Overview
Oil palm fruit bunches progress through distinct stages. Optimal harvest occurs at stage 3 (DARK_RED/peak ripeness).

```
STAGE 0: GREEN (Immature)
├─ Days to optimal: 30 days
├─ Oil content: 8%
├─ FFB weight: ~5 kg
├─ Yield readiness: 0% (not harvestable)
├─ Characteristics:
│  ├─ Fruit: Hard, green color, firmly attached
│  ├─ Fruitlets: Individual fruits still hard to squeeze
│  ├─ Oil content: Minimal (8-12%)
│  ├─ Moisture: ~75%
│  └─ Suitability: Leave on palm

STAGE 1: YELLOW (Early Ripening)
├─ Days to optimal: 20 days
├─ Oil content: 20%
├─ FFB weight: ~15 kg
├─ Yield readiness: 30% (early harvest possible, lower oil)
├─ Characteristics:
│  ├─ Fruit: Some fruits yellow, most still orange-green
│  ├─ Detachment: Some fruitlets drop when pressed (loose)
│  ├─ Oil content: 18-25%
│  ├─ Moisture: ~65%
│  └─ Note: Premature harvest reduces yield 20-30%

STAGE 2: RED (Maturing)
├─ Days to optimal: 10 days
├─ Oil content: 35%
├─ FFB weight: ~22 kg
├─ Yield readiness: 70% (good harvest, acceptable oil content)
├─ Characteristics:
│  ├─ Fruit: Mostly red/orange, 40-50% fruit detached
│  ├─ Detachment: Several fruitlets drop easily
│  ├─ Oil content: 32-38%
│  ├─ Moisture: ~55%
│  └─ Suitability: Can harvest, 1-2 weeks improvement remaining

STAGE 3: DARK_RED (Optimal/Peak Ripeness)
├─ Days to optimal: 0 days (HARVEST NOW)
├─ Oil content: 42%
├─ FFB weight: ~25 kg
├─ Yield readiness: 100% (maximum yield, best quality)
├─ Characteristics:
│  ├─ Fruit: 70-80% red, most others orange
│  ├─ Detachment: Easy detachment - multiple fruitlets fall spontaneously
│  ├─ Oil content: 40-45%
│  ├─ Moisture: ~50%
│  └─ Suitability: ★★★ OPTIMAL HARVEST POINT ★★★

STAGE 4: FALLEN (Over-ripe)
├─ Days to optimal: -1 days (PAST OPTIMAL)
├─ Oil content: 40% (slight degradation)
├─ FFB weight: ~24 kg (losses from ground time)
├─ Yield readiness: 95% (quality risk: fungal infection, oxidation)
├─ Characteristics:
│  ├─ Fruit: Fully dark red/almost black
│  ├─ Status: Bunch has fallen to ground naturally
│  ├─ Oil content: 38-42% (some degradation)
│  ├─ Moisture: ~48%
│  ├─ Risk factors:
│  │  ├─ Fungal infection from soil contact (Fusarium, Rhizopus)
│  │  ├─ Oxidation and rancidity
│  │  ├─ Pest damage/insect infestation
│  │  └─ Reduced oil quality (FFA increases)
│  └─ Action: Collect quickly, process same day to minimize loss
```

## 2. Visual Ripeness Assessment Guide

### Color Progression (Photo Reference)
```
GREEN (Stage 0)
└─ All fruitlets bright green
   └─ No fruit drop
   └─ Bunch firmly attached
   └─ Hard individual fruits

YELLOW (Stage 1)
└─ First fruitlets turning yellow/orange
   └─ 10-20% of bunch showing color
   └─ Tight, minimal drop
   └─ Heterogeneous coloring

RED (Stage 2)
└─ 40-60% of bunch red
   └─ Significant color variation
   └─ Some natural fruit drop beginning
   └─ Red fruitlets easily detachable

DARK RED (Stage 3) ★ TARGET
└─ 70-90% dark red/maroon
   └─ Uniform deep coloring
   └─ Many fruitlets drop at slight touch
   └─ Rich, dark appearance
   └─ Peak harvest window

FALLEN (Stage 4)
└─ On ground, very dark red/near black
   └─ Heavy natural fruit drop
   └─ Potential soil contact/contamination
   └─ Large gaps visible in bunch rachis
```

## 3. Oil Content Yield Calculations

### Oil Yield Formula
```
CRUDE PALM OIL EXTRACTION YIELD:

Crude Palm Oil (CPO) = FFB × Oil Content % × Extraction Efficiency
                     = FFB × Oil % × (85-92%)

Example Calculation:
─────────────────────────────────────────────────────────

Stage 3 (DARK_RED) Optimal:
├─ Fresh Fruit Bunch (FFB) weight: 25 kg
├─ Oil content: 42%
├─ Extraction efficiency: 88%
├─ Crude Palm Oil: 25 × 0.42 × 0.88 = 9.24 kg CPO per bunch

Annual Plantation Yield (1,000 palms):
├─ Bunches/palm/cycle: 8-12 bunches (use 10)
├─ Cycles/year: 2
├─ Total bunches/year: 1,000 × 10 × 2 = 20,000 bunches
├─ Total CPO: 20,000 × 9.24 = 184,800 kg = 184.8 MT/year

Yield per hectare (120 palms/ha standard):
├─ Plantation: 1,000 palms = 8.3 hectares
├─ Annual yield: 184.8 MT ÷ 8.3 ha = 22.3 MT/ha/year
└─ Benchmark: 18-22 MT/ha/year (excellent), >25 MT/ha (outstanding)
```

### Oil Content Variance by Stage
```
Oil Content Progression:

Stage 0 GREEN:    8% oil
                  └─ NOT ECONOMICAL (high moisture loss during processing)

Stage 1 YELLOW:   20% oil
                  └─ SUBOPTIMAL (20-30% lower yield than optimal)
                  └─ Oil quality issues: Higher FFA (Free Fatty Acid)

Stage 2 RED:      35% oil
                  └─ ACCEPTABLE (70-85% of optimal yield)
                  └─ Trade-off: 10% yield loss for convenience

Stage 3 DARK_RED: 42% oil ★ TARGET FOR PEAK YIELD ★
                  └─ OPTIMAL (100% reference)
                  └─ Best oil quality: FFA < 3%

Stage 4 FALLEN:   40% oil
                  └─ EQUIVALENT OIL (slight 5% loss from oxidation)
                  └─ BUT: Quality degradation
                     ├─ FFA increases to 4-6%
                     ├─ Peroxide value increases
                     └─ Requires immediate processing (risk of rancidity)
```

### Comparative Yield Loss by Stage
```
Reference Yield Comparison (Stage 3 = 100%):

Stage 0 GREEN:
├─ CPO yield: 19% of optimal
├─ Processing loss: Not recommended
└─ Recommendation: ✗ DO NOT HARVEST

Stage 1 YELLOW:
├─ CPO yield: 48% of optimal
├─ Loss vs. Stage 3: 52% LOSS
├─ Time to optimal: 20 days
└─ Recommendation: ⚠️ WAIT 2-3 WEEKS for better yield

├─ Convenience factor: Can harvest now vs. wait 1 week
└─ Recommendation: ✓ ACCEPTABLE (if labor constraints)

Stage 3 DARK_RED:
├─ CPO yield: 100% (reference)
├─ Loss vs. optimal: 0%
├─ Peak ripeness window: 5-7 days
├─ Oil quality: Best (FFA < 3%)
└─ Recommendation: ★ OPTIMAL HARVEST POINT ★

Stage 4 FALLEN:
├─ CPO yield: 95% of optimal
├─ Loss vs. Stage 3: 5% (slight)
├─ BUT quality issues: FFA increases 4-6%
├─ Processing time: URGENT (same day)
├─ Risk of loss: Fungal infection (Fusarium, Rhizopus)
└─ Recommendation: ⚠️ COLLECT IMMEDIATELY (high quality loss risk)
```

## 5. Yield Prediction by Ripeness Distribution

### Plantation-Level Prediction Model

```
Given: Ripeness audit of 100 bunches sampled across 1,000-palm plantation

Input Example:
├─ Stage 0 (GREEN):     10 bunches
├─ Stage 1 (YELLOW):    15 bunches
├─ Stage 2 (RED):       35 bunches
├─ Stage 3 (DARK_RED):  30 bunches (★ harvestable)
├─ Stage 4 (FALLEN):     5 bunches
└─ Unknown:              5 bunches

Harvestable Bunches (Stage 2-4): 70 bunches = 70%

CPO Yield Calculation:
───────────────────────────────

Stage 2 (RED):
├─ Bunches: 35
├─ Weight: 22 kg average
├─ Oil content: 35%
├─ CPO per bunch: 22 × 0.35 × 0.88 = 6.78 kg
├─ Subtotal CPO: 35 × 6.78 = 237 kg

Stage 3 (DARK_RED):
├─ Bunches: 30
├─ Weight: 25 kg average
├─ Oil content: 42%
├─ CPO per bunch: 25 × 0.42 × 0.88 = 9.24 kg
├─ Subtotal CPO: 30 × 9.24 = 277 kg

Stage 4 (FALLEN):
├─ Bunches: 5
├─ Weight: 24 kg average
├─ Oil content: 40% (quality loss accounted)
├─ CPO per bunch: 24 × 0.40 × 0.88 = 8.45 kg
├─ Subtotal CPO: 5 × 8.45 = 42 kg

Total Sample CPO: 237 + 277 + 42 = 556 kg (from 70 harvestable bunches)

Extrapolation to 1,000-Palm Plantation:
────────────────────────────────────────

Assumptions:
├─ 10 harvestable bunches per palm (conservative)
├─ Total harvestable bunches: 1,000 × 10 × 0.70 = 7,000 bunches
│  (70% ready across plantation based on sampling)
│
├─ Average CPO per harvestable bunch: 556/70 = 7.94 kg
├─ Total CPO: 7,000 × 7.94 = 55,580 kg = 55.6 MT

Annual Projection (2 cycles/year):
├─ Cycle 1: 55.6 MT
├─ Cycle 2: 55.6 MT (assuming similar ripeness distribution)
├─ Annual Total: 111.2 MT/year
└─ Per hectare: 111.2 / 8.3 ha = 13.4 MT/ha/year

Quality Metrics:
├─ Average FFA (Free Fatty Acid): 2.8% (good, <3% threshold)
├─ Processing efficiency: 88% (standard)
├─ Market grade: Premium (if harvest at Stage 3 dominates)
└─ Expected price premium: +5-8% over standard
```

## 6. Harvest Scheduling Optimization

### Optimal Harvest Sequence
```
Plantation Strategy for Maximum Annual Yield:

Month-by-Month Projection (2 cycles/year):

Cycle 1 (Months 1-3):
├─ Month 0: Initial survey → 30% Stage 3, 35% Stage 2
├─ Month 1:
│  ├─ Harvest all Stage 3 + Stage 4 bunches (immediate action)
│  ├─ Expected yield: 35% × 9.24 kg + 5% × 8.45 kg = 3.24 + 0.42 = 3.66 kg/palm
│  ├─ Plantation yield: 1,000 × 3.66 = 3,660 kg
│  └─ Action: 40% of plantation harvested
│
├─ Month 1-2:
│  ├─ Stage 2 (RED) maturing to Stage 3
│  ├─ Revisit in 1 week
│  └─ Second harvest wave
│
├─ Month 2-3:
│  ├─ Harvest Stage 2 → Stage 3 bunches
│  ├─ Expected yield: 35% × 9.24 kg = 3.23 kg/palm
│  ├─ Plantation yield: 1,000 × 3.23 = 3,230 kg
│  └─ Action: 35% of plantation harvested (total 75%)
│
└─ Cycle 1 Total: ~6,890 kg CPO (6.9 MT)

Cycle 2 (Months 4-6):
├─ Regeneration: New bunches developing on Stage 1-2
├─ Timing: 5-6 weeks after first cycle harvest
├─ Expected yield: Similar distribution
└─ Total: ~6,890 kg CPO

Annual Total: 13.78 MT (13.78 / 8.3 ha = 1.66 MT/ha/cycle or 3.32 MT/ha/year)

⚠️ Note: Low yield example; better management targets 18-22 MT/ha/year
```

```
Week 2: Zone B survey → 35% harvestable
├─ Harvest Zone B Stage 3-4

└─ Manages transport/processing capacity uniformly
```

## 7. Ripeness Assessment Checklist for Farmers

### Field Inspection Procedure
```
Time: 10-15 minutes per 100 palms
Equipment: Harvest knife, color reference chart, sampling bags

Step 1: Visual Survey
├─ Walk plantation systematically
├─ Assess 1-2 bunches per palm in test row
├─ Count bunches by stage (visual classification)
└─ Record: Date, Location, Stage distribution

Step 2: Touch Test
├─ Gently squeeze individual fruitlets
├─ EASY detachment = Stage 3 (ready)
├─ SLIGHT detachment = Stage 2 (wait 1 week)
├─ NO detachment = Stage 1-0 (wait 2+ weeks)
└─ Note: Don't damage fruit during testing

Step 3: Color Assessment
├─ Compare to reference: GREEN → YELLOW → RED → DARK_RED
├─ Percentage rule: >50% red = Stage 2 minimum
├─ >70% dark red = Stage 3 (optimal)
└─ On ground = Stage 4 (collect same day)

Step 4: Weight Estimate
├─ Mature bunches: 20-25 kg (Stage 2-3)
├─ Immature: <15 kg (Stage 0-1)
├─ If unknown: Request harvest assistance for assessment

Step 5: Recording
├─ Fill ripeness monitoring form:
│  ├─ Date: __/__/____
│  ├─ Location/Zone: ___________
│  ├─ Stage 0 (GREEN): ___ bunches
│  ├─ Stage 1 (YELLOW): ___ bunches
│  ├─ Stage 2 (RED): ___ bunches
│  ├─ Stage 3 (DARK_RED): ___ bunches
│  ├─ Stage 4 (FALLEN): ___ bunches
│  ├─ Recommendation: [WAIT / HARVEST SOON / HARVEST NOW]
│  └─ Notes: _____________________
│
└─ Share with harvest supervisor for scheduling
```

### Harvest Readiness Criteria
```
✓ READY TO HARVEST:
├─ Stage 3 bunches visible (>20% of sampled)
├─ Stage 4 bunches present (collect immediately)
├─ Touch test: Easy fruitlet detachment
├─ Recommendation: HARVEST TODAY

⚠️  HARVEST IN 1-2 WEEKS:
├─ Stage 2 predominant (>50% of sample)
├─ Stage 3 emerging (<20%)
├─ Touch test: Slight fruitlet detachment
├─ Recommendation: WAIT 7-10 DAYS

✗ NOT READY (WAIT 2+ WEEKS):
├─ Mostly Stage 1 bunches
├─ Stage 0 bunches present (>30%)
├─ Touch test: No fruitlet detachment
├─ Recommendation: WAIT 3-4 WEEKS, REVISIT
```
